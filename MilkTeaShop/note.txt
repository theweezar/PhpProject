Get list of command => php artisan list
Get help of method => php artisan help <method>
Deploy app => php artisan serve
Create a controller => "php artisan make:controller <name>"
Create a model => "php artisan make:model <name>"

clone model from database => https://github.com/krlove/eloquent-model-generator
+ php artisan krlove:generate:model User
+ php artisan krlove:generate:model Client --table-name=client => clone table tên "client" vào folder app với tên Client









config debug mode => config/app.php
config database => config/database.php (database, username, password)
